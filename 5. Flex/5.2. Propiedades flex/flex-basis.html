La propiedad flex-basis establece el tamaño inicial de los ítems flexibles antes de que el espacio disponible se reparta entre ellos (por ejemplo, usando flex-grow). Es decir, define el tamaño base que tendrá cada elemento dentro del contenedor flex.

Según la especificación, flex-basis funciona de forma muy similar a la propiedad width, pero existen algunas diferencias importantes cuando se usan los valores auto y content:

  • Si el valor es auto, el tamaño base del ítem será el que indique la propiedad width. Si width también es auto, entonces el tamaño se calculará automáticamente según el contenido del elemento (como si fuera content).
  • Si el valor es content, el tamaño base será exactamente el tamaño del contenido del ítem flexible, sin recortes ni expansiones. En la práctica, suele comportarse igual que max-content, es decir, el ancho mínimo necesario para mostrar todo el contenido en una sola línea.
