La propiedad flex es un atajo que combina las propiedades flex-grow, flex-shrink y flex-basis en una sola declaración. Su sintaxis es: flex: [flex-grow] [flex-shrink] [flex-basis].

Los valores especiales de flex son:

  • flex: initial (valor por defecto). Equivale a flex: 0 1 auto. El elemento no crece (flex-grow: 0), pero puede encogerse (flex-shrink: 1) si es necesario. Su tamaño base es automático (flex-basis: auto).
    Con width establecido: Los elementos mantienen su width definido (150px, 200px, 250px) y solo se encogen si el contenedor es demasiado pequeño para contenerlos.
    Sin width: Los elementos toman el tamaño de su contenido y se encogen proporcionalmente si es necesario para caber en el contenedor.
  • flex: none. Equivale a flex: 0 0 auto. El elemento no crece ni se encoge (flex-grow: 0 y flex-shrink: 0), manteniendo siempre su tamaño natural. Es completamente inflexible.
    Con width establecido: Los elementos mantienen estrictamente su width definido (150px, 200px, 250px) sin cambios, pudiendo causar desbordamiento si el contenedor es más pequeño.
    Sin width: Los elementos mantienen exactamente el tamaño de su contenido sin modificaciones, pudiendo desbordarse si el contenido es muy largo para el contenedor.
  • flex: auto. Equivale a flex: 1 1 auto. El elemento puede crecer y encogerse (flex-grow: 1 y flex-shrink: 1), usando su contenido como tamaño base (flex-basis: auto). Los elementos se distribuyen el espacio extra proporcionalmente a su tamaño inicial.
    Con width establecido: Los elementos usan su width definido como base y se distribuyen el espacio restante proporcionalmente, manteniendo sus diferencias de tamaño relativas.
    Sin width: Los elementos usan el tamaño de su contenido como base y se distribuyen el espacio extra proporcionalmente a su contenido inicial, los elementos con más contenido crecen más.
  • flex: 1. Equivale a flex: 1 1 0%. El elemento puede crecer y encogerse (flex-grow: 1 y flex-shrink: 1), pero su tamaño base es 0 (flex-basis: 0%). Los elementos se distribuyen todo el espacio disponible de forma equitativa.
    El valor de width aquí es irrelevante, ya que se ignora debido a flex-basis: 0%. Los elementos se distribuyen el espacio equitativamente (cada uno ocupa 1/3 del contenedor). Pero, cuidado, lo que distribuye equitativamente flex: 1 es el content-box de los elementos, no el tamaño total incluyendo padding y border. Por eso, si los elementos tienen diferentes paddings o borders, el espacio visual ocupado puede variar ligeramente. Puedes comprobar este efecto añadiendo un padding a uno de los ítems.
  • flex: 0. Equivale a flex: 0 1 0%. El elemento no crece (flex-grow: 0), pero puede encogerse (flex-shrink: 1) si es necesario. Su tamaño base es 0 (flex-basis: 0%), por lo que depende completamente de su contenido.
    El valor de width aquí es irrelevante, ya que se ignora debido a flex-basis: 0%. Los elementos toman el tamaño mínimo necesario para su contenido y se encogen proporcionalmente si el contenedor es muy pequeño.

Nota: La diferencia clave entre flex: auto y flex: 1 está en el flex-basis: auto respeta el tamaño del contenido como base, mientras que flex: 1 ignora el contenido (basis: 0%) y distribuye equitativamente. Entre flex: initial y flex: 0 la diferencia está en el flex-basis: initial usa el tamaño natural (auto) mientras que 0 parte de tamaño cero y se ajusta al contenido.
