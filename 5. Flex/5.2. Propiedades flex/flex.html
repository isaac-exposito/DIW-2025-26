La propiedad flex es un atajo que combina las propiedades flex-grow, flex-shrink y flex-basis en una sola declaración. Su sintaxis es: flex: [flex-grow] [flex-shrink] [flex-basis].

Los valores especiales de flex son:

  • flex: initial (valor por defecto). Equivale a poner cada una de las 3 propiedades a su valor por defecto, es decir, flex: 0 1 auto. El elemento no crece (flex-grow: 0), pero puede encogerse (flex-shrink: 1) si es necesario. Su tamaño base es automático (flex-basis: auto).
    Con width establecido: Los elementos mantienen su width definido (150px, 200px, 250px), pero se encogerán si es necesario.
    Sin width: Los elementos toman el tamaño de su contenido y se encogen proporcionalmente si es necesario.
  • flex: none. Equivale a flex: 0 0 auto. El elemento no crece ni se encoge, manteniendo su tamaño natural. Es completamente inflexible.
    Con width establecido: Los elementos mantienen estrictamente su width definido (150px, 200px, 250px) sin cambios, pudiendo causar desbordamiento.
    Sin width: Los elementos mantienen el tamaño de su contenido sin cambios, pudiendo causar desbordamiento si el contenido es muy largo.
  • flex: auto. Equivale a flex: 1 1 auto. El elemento puede crecer y encogerse, pero su tamaño base es automático (basado en su contenido). Los elementos se ajustan proporcionalmente a su contenido inicial.
    Con width establecido: Los elementos parten del width definido como base y crecen/encogen proporcionalmente, manteniendo sus diferencias de tamaño relativas.
    Sin width: Los elementos parten del tamaño de su contenido como base y se distribuyen el espacio extra proporcionalmente a su contenido inicial.
  • flex: 1. Equivale a flex: 1 1 0%. El elemento puede crecer y encogerse con un factor de 1, y su tamaño base es 0. Los elementos se distribuyen el espacio disponible equitativamente.
    Con width establecido: A pesar del width definido, los elementos se distribuyen el espacio total equitativamente (cada uno ocupa 1/3 del contenedor).
    Sin width: Los elementos se distribuyen el espacio disponible de forma totalmente equitativa, ignorando las diferencias de contenido.
  • flex: 0. Equivale a flex: 0 1 0%. El elemento no puede crecer (flex-grow: 0), pero puede encogerse (flex-shrink: 1) si es necesario, y su tamaño base es 0 (flex-basis: 0%). Los elementos se ajustan únicamente a su contenido mínimo.
    Con width establecido: Los elementos ignoran el width definido y se ajustan al tamaño mínimo necesario para su contenido, sin crecer aunque haya espacio disponible.
    Sin width: Los elementos se ajustan al tamaño mínimo de su contenido y no crecen, ocupando solo el espacio estrictamente necesario.
