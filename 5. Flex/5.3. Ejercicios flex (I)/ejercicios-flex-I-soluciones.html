Ejercicios Flex (I) - Soluciones

Partiendo del HTML y CSS de ejercicios-flex-I-codigo.html realiza los siguientes diseños utilizando las propiedades de los elementos flex. En los casos en los que se pueda hacer de varias maneras, busca la que menos propiedades haya que utilizar.

Ver solo enunciados

Ejercicio 1
Solución:
main {
  flex-direction: row-reverse;
}
Ejercicio 2
Solución:
main {
  flex-direction: column;
}
