Hacer esta calculadora con números de líneas es extremadamente simple. Podríamos hacerlo de varias maneras:

La primera, la más enrevesada, es usar las 4 propiedades en cada uno de los 30 items, indicando para cada uno en qué línea empieza y acaba en filas y columnas. El primer elemento sería:

.mc {
  grid-row-start: 1;
  grid-row-end: 2;
  grid-column-start: 1;
  grid-column-end: 3;
}

Eso serían 30*4=120 líneas de CSS. Podríamos reducirlas a la mitad eliminando las que se refieren a filas, dado que el navegador irá insertando los ítems en filas automáticamente. Así, el primer ítem quedaría:

.mc {
  grid-column-start: 1;
  grid-column-end: 3;
}

Pero podemos seguir reduciendo las líneas necesarias. Dado que el navegador va insertando los items consecutivamente, no necesitamos indicarle dónde debe empezar cada ítem, nos basta con indicarle que ocupará X columnas, por lo que nos sobra grid-column-start:

.mc {
  grid-column-end: span 2;
}

Incluso podemos deshacernos de las 30 clases que hemos creado solo para indicar las líneas, simplemente añadiendo dos clases nuevas y asignándoselas a cada ítem:

.span-2 {
  grid-column-end: span 2;
}

.span-3 {
  grid-column-end: span 3;
}
<div class="contenedor-calculadora">
  <div class="item span-2">MC</div>
  <div class="item span-2">MR</div>
  ...
  <div class="item span-3">,</div>
  <div class="item span-3">=</div>
</div>

Por último, podemos incluso olvidarnos de crear clases y usar selectores más finos para los elementos, de manera que seleccionemos los 6 primeros por un lado y los restantes por otro:

/* Los seis primeros hijos con clase .item ocuparán 2 columnas */
.grid>.item:nth-child(-n+6) {
  grid-column-end: span 2;
}

/* Los .item hijos a partir del séptimo (incluido) ocuparán 3 columnas */
.grid>.item:nth-child(n+7) {
  grid-column-end: span 3;
}
