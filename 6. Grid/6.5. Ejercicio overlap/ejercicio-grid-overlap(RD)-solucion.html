Grid overlap

Añade una media query para que a partir de 600px el texto se superponga a la imagen. El texto debe estar situado aproximadamente en el centro de la imagen tanto vertical como horizontalmente.

Explicación

Para la estructura del grid definimos un grid de 4 columnas con grid-template-columns: repeat(4, auto). El valor auto hace que cada columna se ajuste al contenido. Podríamos usar fr pero eso, en este caso particular, nos obligartía a aumentar el número de columnas para que la imagen quedara centrada y visible. Puedes probar con repeat(9, 1fr).

Posicionamiento de la imagen

  • Filas: grid-row: 1 / -1. Solo hay una fila, pero indicamos que la imagen ocupe desde la primera fila hasta la última.
  • Columnas: grid-column: 1 / 3. La imagen ocupa las columnas 1 y 2.

Posicionamiento del texto

  • Filas: grid-row: 1 / -1. Igual que en el caso anterior, el texto ocupa la única fila creada.
  • Columnas: grid-column: 2 / -1. Indicamos que el texto ocupe desde la columna 2 hasta la última.
  • Alineación: align-self: center. Centra verticalmente el texto dentro de su celda.
