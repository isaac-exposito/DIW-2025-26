Explicación:

En el header hay que poner grid-auto-flow: column y justify-content: space-between. Para poner el logo a la derecha simplemente order: 1.

En la parte central, si queremos ser exquisitos con el layout original con float, deberíamos poner grid-template-columns: 25% 70%, pero mejor 1fr 3fr. La separación entre columnas se puede establecer con column-gap. Para las filas sería grid-template-rows: 150px 150px para que se estiren las filas y quede como en el original. Para que el sidebar ocupe la primera columna, simplemente habría que añadirle grid-row: 1 / 3.
